# Website_ORION
Desenvolvimento de um website institucional completo para uma empresa de terceirização de serviços, com foco em apresentar de forma clara os serviços oferecidos, fortalecer a credibilidade da marca e facilitar o contato com clientes corporativos.

O projeto conta com design moderno e responsivo, navegação intuitiva e estrutura pensada para conversão, incluindo seções de serviços, diferenciais competitivos, depoimentos, formulário de contato e chamadas estratégicas para solicitação de orçamento.

Tecnologias utilizadas: HTML, CSS e JavaScript, com atenção especial à performance, acessibilidade e experiência do usuário em dispositivos desktop e mobile.
